Real-Time Monitoring and Detection of Neuromuscular Diseases

Developed a low-cost (<INR 500) point-of-care system for early ALS detection using surface EMG signals. Integrated custom hardware and wireless transmission to a remote server. Used a modified VGG-5 (SpinaINet-VGG) to classify EMG spectrograms with 96.08% accuracy.

Medical Diagnosis using Genetic Algorithm and Neural Networks

Applied a genetic algorithm (pop. size 20, crossover 0.4, mutation 0.055) for feature selection on 1,098 gastrointestinal bleeding WCE images using pre-trained CNNs (DenseNet, ResNet, InceptionV3). Ensemble model achieved 92% accuracy, outperforming individual models (74–76%, improved to 88–90% with data augmentation).
